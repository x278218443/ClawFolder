## 飞书文件发送限制（重要！）
- 飞书 message 工具 `filePath` 参数：**<30MB 可以发文件附件**，>30MB 只显示路径文本
- 7.2MB Seedance 测试视频 ✅ 正常发送并播放
- 26MB 压缩视频 ✅ 正常发送（CRF 28 压缩）
- 57MB 原始视频 ❌ 只显示链接
- `buffer`（base64）方式：小文件可行，大文件（>10MB）不实际
- `media` 参数效果同 `filePath`
- **飞书文件上传 API 硬限制 30MB**
